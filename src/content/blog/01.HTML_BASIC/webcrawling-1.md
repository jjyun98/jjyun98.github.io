## 1. 웹 개발의 3요소

- **HTML**: 웹 페이지의 **구조(structure)** 를 담당  
- **CSS**: 웹 페이지의 **디자인(style)** 을 담당  
- **JavaScript**: 웹 페이지의 **동작(behavior)** 을 담당  

## 2. HTML 기본 구조

HTML 문서는 **태그(tag)** 를 이용해 구조를 정의한다.  
앞에 나온 태그를 시작태그, 뒤에 나온 태그를 종료태그라고 한다.

```html
<태그이름>내용</태그이름>
```

```html
<태그이름 속성="속성값">내용</태그이름>
```

- **속성(attribute)**: 태그에 추가 정보를 제공
- 속성은 **여러 개** 사용 가능
- 속성이 **없어도** 됨
