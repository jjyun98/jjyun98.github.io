## HTML 

> 웹 사이트 개발의 3요소
    HTML : 구조
    CSS : 디자인
    JavaScript : 동작


> HTML 구조
    태그 구조
    <태그이름>내용</태그이름>
    앞 태그 이름=시작태그
    뒤 태그 이름=종료태그

> 태그 구조
    <태그이름 속성="속성값">내용</태그이름>

- 속성(attribute): 태그의 추가적인 정보
- 속성은 여러 개 부여할 수 있다.
- 속성은 없어도 된다.

![alt text](/assets/images/crawling0.png)

예시를 보면 해드 태그(1) ... 해드 태그(2)
1번이 시작태그, 2번이 종료태그가 된다.

그 아래 바디 태그도 내용이 많지만 똑같다.
주황색 글자들은 속성을 의미한다. 속성은 태그 이름 뒤에 적어주는데 속성은 태그의 추가적인 정보를 제공하는 역할을 한다.
예시를 보면 바디 태그 안의 div 태그의 id라는 태그 속성의 속성 값이 "u_skip"이 있다고 할 수 있다.

- 내용에는 텍스트나 태그가 들어갈 수 있다.
- 내용은 없어도 된다.

![alt text](/assets/images/crawling1.png)

예시를 보면 wrap 속성을 가진 div 태그 안에 script라는 내용이 3개가 들어 있다.

이들의 관계를 부모-자식 태그라고 부른다. 위의
div id="wrap" : 부모태그
script type ... : 자식태그


```html
<!DOCTYPE html>
<head>
    <!-- 부가적인 정보(웹 사이트 제목, 설명, 메타태그) -->
</head>
<body>
    <!-- 실제로 화면에 표시되는 내용 -->
    
</body>
</html>
```

html문서 전체 구조를 보면 <!DOCTYPE html>은 현재 HTML의 버전을 알려주는 것이고, 이것은 현재 HTML5로 작성됨을 알려주고 있다.
그 다음 head태그와 body태그로 나뉠수 있는데 head태그에는 부가적인 정보가 들어가고, body태그 안에 화면에 표시되는 내용이 들어간다. 크롤링 시에도 이 부분을 살펴보게 된다.

주석(comments)

주석은 다음과 같이 사용한다.

```html
<!-- 주석1 -->
<!-- <p>주석2</p> -->
```
